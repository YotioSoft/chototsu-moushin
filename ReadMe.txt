1. はじめに
　この度は、「猪突猛進」をダウンロードしていただきありがとうございま

2. 概要
　スペースキーを連打して、イノシシを止めてください。崖に落ちるまでに
止められないと、ゲームオーバーです。

3. 操作方法
SPACE : イノシシを止める
